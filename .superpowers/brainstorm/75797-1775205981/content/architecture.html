Section 1 of 4: How the pieces fit together

Pipeline Flow
scanner.ts → scoring.ts → briefing.ts → BRIEFING.md (unchanged)
↘ registry.json → ai-briefing.ts + prompts/briefing.md → telegram.ts

New Files

src/ai-briefing.ts
  • Loads prompt template from disk
  • Injects registry.json as context
  • Calls Claude Haiku API
  • Returns formatted Telegram HTML
prompts/briefing.md
  • System prompt for briefing generation
  • Defines tone, format, structure
  • Instructs cross-project analysis
  • Specifies Telegram HTML output format
Modified Files
  • orchestrator.config.ts — add AI config
  • src/types.ts — add AIConfig type
  • src/cli.ts — swap in AI briefing for notify
